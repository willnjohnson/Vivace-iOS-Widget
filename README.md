# Vivace iOS Widget

Vivace (iOS) is a lightweight **iOS widget for Scriptable** that displays the a unique calendar directly on your **Home Screen or Lock Screen**.

This widget is based on my desktop **Vivace app**:  
https://github.com/willnjohnson/Vivace

## Features

- Displays the French Revolutionary date
  - Month
  - Day
  - Year
  - Emoji (corresponding to the month)
  - Item name (corresponding to the day)
- Works as a **Scriptable widget**
- Can be added to:
  - Home Screen
  - Lock Screen (iOS 16+)

## Requirements

- iOS 15 or later (Lock Screen widgets require iOS 16+)
- [Scriptable](https://apps.apple.com/app/scriptable/id1405459188) installed on your device

## Installation

1. **Install Scriptable**
   - Download Scriptable from the App Store

2. **Add the Script**
   - Open Scriptable
   - Create a new script
   - Copy and paste the source code from this repository into the script
   - Save it (e.g. `Vivace iOS Widget`)

3. **Run Once (Recommended)**
   - Run the script once inside Scriptable to ensure permissions are granted

## Adding the Widget

### Home Screen

1. Long-press on your Home Screen
2. Tap the **+** button
3. Search for **Scriptable**
4. Choose a widget size
5. Tap **Add Widget**

### Lock Screen (iOS 16+)

1. Long-press the Lock Screen
2. Tap **Customize**
3. Select **Lock Screen**
4. Tap the widget area
5. Choose **Scriptable**
6. Select the Vivace script
